## Usage

```tsx
import { RadioGroup } from "@adn-ui/react";

export default () => (
  <RadioGroup.Root defaultValue="1">
    <RadioGroup.Item value="1">
      <RadioGroup.Indicator />
    </RadioGroup.Item>
    <RadioGroup.Item value="2">
      <RadioGroup.Indicator />
    </RadioGroup.Item>
  </RadioGroup.Root>
);
```

## API Reference

### Root
The foundational component for the radio group.

### Item
The checkable button.

### Indicator
Renders when the radio button is checked.
