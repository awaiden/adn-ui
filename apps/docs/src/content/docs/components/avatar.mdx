An image element with a fallback for representing the user.

## Preview

<ComponentPreview>
  <AvatarExample.Default client:load />
</ComponentPreview>

## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback } from "@adn-ui/react";

export default () => (
  <Avatar>
    <AvatarImage src="https://github.com/mdo.png" alt="@mdo" />
    <AvatarFallback>M</AvatarFallback>
  </Avatar>
);
```

The image to render. By default it will only render when it has loaded.

Element that renders when the image hasn't loaded. This can be used for initials, icons or any other placeholder.
