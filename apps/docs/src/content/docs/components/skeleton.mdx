## Preview

<ComponentPreview>
  <SkeletonExample.Default client:load />
</ComponentPreview>

## Usage

```tsx
import { Skeleton } from "@adn-ui/react";

export default () => <Skeleton className="h-4 w-[250px]" />;
```

## Examples

### Default
A rectangular skeleton with a fixed width and height.
<ComponentPreview>
  <SkeletonExample.Default client:load />
</ComponentPreview>

```tsx
<Skeleton className="h-4 w-[250px]" />
```

### Circular
A circular skeleton used to represent avatars or icons.
<ComponentPreview>
  <SkeletonExample.Circular client:load />
</ComponentPreview>

```tsx
<Skeleton variant="circular" className="size-12" />
```

### Card Skeleton
Combine circular and rectangular skeletons to mock a card layout.
<ComponentPreview>
  <SkeletonExample.Card client:load />
</ComponentPreview>

```tsx
<div className="flex items-center space-x-4">
  <Skeleton variant="circular" className="size-12" />
  <div className="space-y-2">
    <Skeleton className="h-4 w-[250px]" />
    <Skeleton className="h-4 w-[200px]" />
  </div>
</div>
```

### Form Skeleton
Mock a form layout while content loads.
<ComponentPreview>
  <SkeletonExample.Form client:load />
</ComponentPreview>

```tsx
<div className="space-y-4 w-[350px]">
  <div className="space-y-2">
    <Skeleton className="h-4 w-[80px]" />
    <Skeleton className="h-10 w-full" />
  </div>
  <div className="space-y-2">
    <Skeleton className="h-4 w-[120px]" />
    <Skeleton className="h-10 w-full" />
  </div>
  <Skeleton className="h-10 w-[100px]" />
</div>
```
