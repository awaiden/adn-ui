<ComponentPreview>

</ComponentPreview>

## Usage

```tsx
import { Tooltip, Button } from "@adn-ui/react";

export default () => (
  <Tooltip.Provider>
    <Tooltip.Root>
      <Tooltip.Trigger asChild>
        <Button variant="outline">Hover me</Button>
      </Tooltip.Trigger>
      <Tooltip.Content>
        <p>Add to library</p>
      </Tooltip.Content>
    </Tooltip.Root>
  </Tooltip.Provider>
);
```

## API Reference

### Provider

Wraps your app (or a specific section) to provide tooltip functionality.

### Root

Contains all the parts of an individual tooltip.

### Trigger

The element that triggers the tooltip.
