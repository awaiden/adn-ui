Displays a list of options for the user to pick from—triggered by a button.

## Preview

<ComponentPreview>
  <SelectExample.Default client:load />
</ComponentPreview>

## Usage

```tsx
import { Select } from "@adn-ui/react";

export default () => (
  <Select.Root>
    <Select.Trigger>
      <Select.Value placeholder="Select a fruit" />
    </Select.Trigger>
    <Select.Content>
      <Select.Item value="apple">Apple</Select.Item>
      <Select.Item value="banana">Banana</Select.Item>
    </Select.Content>
  </Select.Root>
);
```

### Trigger

The button that toggles the select.

### Value

The value displayed in the trigger when an item is selected.

### Content

Contains the items to be displayed in the select.

### Item

A single option in the select.

### Group

Used to group multiple items together.

### Label

Used to render a label for a group.
