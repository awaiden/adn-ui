Displays a button or a component that looks like a button.

## Preview

<ComponentPreview>
  <ButtonExample.Variants client:load />
</ComponentPreview>

## Usage

```tsx
import { Button } from "@adn-ui/react";

export default () => (
  <Button variant="primary" size="md">
    Click me
  </Button>
);
```

## Examples

### Variants

Available variants include primary, secondary, outline, ghost, and destructive.

<ComponentPreview>
  <ButtonExample.Variants client:load />
</ComponentPreview>

```tsx
<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="destructive">Destructive</Button>
```

### Sizes

Available sizes include small, medium, large, and an icon size.

<ComponentPreview>
  <ButtonExample.Sizes client:load />
</ComponentPreview>

```tsx
<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>
<Button size="icon">🔍</Button>
```

### Disabled

You can disable the button by passing the `disabled` prop.

<ComponentPreview>
  <ButtonExample.Disabled client:load />
</ComponentPreview>

```tsx
<Button disabled>Disabled Button</Button>
```
