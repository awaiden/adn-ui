## Usage

```tsx
import { Collapsible } from "@adn-ui/react";

export default () => (
  <Collapsible.Root>
    <Collapsible.Trigger>Toggle content</Collapsible.Trigger>
    <Collapsible.Content>
      This is the collapsible content.
    </Collapsible.Content>
  </Collapsible.Root>
);
```

### Trigger
The button that toggles the collapsible.

### Content
The content that is revealed when the collapsible is open.
