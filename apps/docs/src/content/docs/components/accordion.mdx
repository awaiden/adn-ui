A vertically stacked set of interactive headings that each reveal a section of content.

## Preview

<ComponentPreview>
  <AccordionExample.Single client:load />
</ComponentPreview>

## Usage

```tsx
import { Accordion } from "@adn-ui/react";

export default () => (
  <Accordion.Root type="single" collapsible>
    <Accordion.Item value="item-1">
      <Accordion.Trigger>Is it accessible?</Accordion.Trigger>
      <Accordion.Content>Yes. It adheres to the WAI-ARIA design pattern.</Accordion.Content>
    </Accordion.Item>
  </Accordion.Root>
);
```

### Item

Contains all the parts of a single collapsible section.

      <td className="py-2 px-4 border-b">`false`</td>
    </tr>
  </tbody>

### Trigger

Toggles the collapsed state of its associated item. It should be nested inside an `Accordion.Header`.

### Content

Contains the collapsible content for an item.
