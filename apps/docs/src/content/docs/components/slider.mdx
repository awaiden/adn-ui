## Preview

<ComponentPreview>
  <SliderExample.Default client:load />
</ComponentPreview>

## Usage

```tsx
import { Slider } from "@adn-ui/react";

export default () => (
  <Slider.Root defaultValue={[50]} max={100} step={1}>
    <Slider.Track>
      <Slider.Range />
    </Slider.Track>
    <Slider.Thumb />
  </Slider.Root>
);
```

### Track

The background track of the slider.

### Range

The active range part of the slider.

### Thumb

A draggable thumb. Render one for each value in `defaultValue` or `value`.
