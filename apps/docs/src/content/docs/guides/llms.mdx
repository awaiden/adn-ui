# LLMs.txt Reference

The [`/llms.txt`](/llms.txt) file provides LLM-friendly, machine-readable documentation for your site. This file is auto-generated and can be accessed at the root of your deployed documentation:

```
https://ui.awaiden.com/llms.txt
```

- Use this file to help large language models (LLMs) understand your documentation structure and content.
- The format is optimized for AI consumption and indexing.

For more details, see the [Starlight LLMs.txt plugin documentation](https://starlight.astro.build/en/guides/llms-txt/).
