## Accessibility Standards

- [WAI-ARIA Authoring Practices](https://www.w3.org/WAI/ARIA/apg/) — The design patterns that Radix UI implements.
- [WCAG 2.1 Guidelines](https://www.w3.org/TR/WCAG21/) — Web Content Accessibility Guidelines that the components conform to.

## Color System

- [OKLCH Color Space](https://oklch.com/) — The perceptually uniform color space used for all design tokens.
- [OKLCH Picker](https://oklch.com/) — Interactive color picker for OKLCH values.

## Related Projects

- [shadcn/ui](https://ui.shadcn.com/) — A similar approach to component libraries using Radix + Tailwind.
- [Radix Themes](https://www.radix-ui.com/themes/docs/overview/getting-started) — Radix's own styled component library.
