## Requirements

- React 19+
- Tailwind CSS 4+ (optional)

## Installation

`@adn-ui/styles` is optional. You can bring your own styles

```package-install
npm install @adn-ui/react @adn-ui/styles
```

## Usage

Import styles

```css
/* globals.css */
@import "tailwindcss";
@import "@adn-ui/styles";
```

Use in components

```tsx
// App.tsx
import { Button } from "@adn-ui/react";
import "@adn-ui/styles";

function App() {
  return <Button>Click Me</Button>;
}
```

That's it! You're now ready to use FeaUI components.
