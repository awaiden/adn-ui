## Styling

### Customizing the component classes

```css
.sidebar {
  @apply bg-background flex h-full flex-col border-r;
}

.sidebar__item {
  @apply hover:bg-muted flex w-full items-center rounded-md p-2 transition-colors;
}
```

## CSS Classes

### Slots

- `sidebar__content`: The class for the content slot.
- `sidebar__footer`: The class for the footer slot.
- `sidebar__header`: The class for the header slot.
- `sidebar__menu`: The class for the menu slot.
- `sidebar__menu-item`: The class for the menuItem slot.
- `sidebar__menu-label`: The class for the menuLabel slot.
- `sidebar__outlet`: The class for the outlet slot.
- `sidebar__panel`: The class for the panel slot.
- `sidebar`: The class for the root slot.
- `sidebar__trigger`: The class for the trigger slot.
