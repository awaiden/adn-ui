## Styling

### Customizing the component classes

```css
.navigation-menu {
  @apply relative z-10 flex max-w-max flex-1 items-center justify-center;
}

.navigation-menu__list {
  @apply group flex flex-1 list-none items-center justify-center space-x-1;
}

.navigation-menu__content {
  @apply data-[motion^=from-]:animate-in data-[motion^=to-]:animate-out data-[motion^=from-]:fade-in data-[motion^=to-]:fade-out data-[motion=from-end]:slide-in-from-right-52 data-[motion=from-start]:slide-in-from-left-52 data-[motion=to-end]:slide-out-to-right-52 data-[motion=to-start]:slide-out-to-left-52 top-0 left-0 w-full md:absolute md:w-auto;
}
```

## CSS Classes

### Slots

- `navigation-menu`: The class for the root slot.
- `navigation-menu__list`: The class for the list slot.
- `navigation-menu__item`: The class for the item slot.
- `navigation-menu__trigger`: The class for the trigger slot.
- `navigation-menu__icon`: The class for the icon slot.
- `navigation-menu__content`: The class for the content slot.
- `navigation-menu__link`: The class for the link slot.
- `navigation-menu__portal`: The class for the portal slot.
- `navigation-menu__backdrop`: The class for the backdrop slot.
- `navigation-menu__positioner`: The class for the positioner slot.
- `navigation-menu__popup`: The class for the popup slot.
- `navigation-menu__arrow`: The class for the arrow slot.
- `navigation-menu__viewport`: The class for the viewport slot.
