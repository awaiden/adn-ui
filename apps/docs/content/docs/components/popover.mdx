## Styling

### Customizing the component classes

```css
.popover__content {
  @apply bg-popover text-popover-foreground data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=closed]:fade-out-0 data-[state=open]:fade-in-0 data-[state=closed]:zoom-out-95 data-[state=open]:zoom-in-95 data-[side=bottom]:slide-in-from-top-2 data-[side=left]:slide-in-from-right-2 data-[side=right]:slide-in-from-left-2 data-[side=top]:slide-in-from-bottom-2 z-50 w-72 rounded-md border p-4 shadow-md outline-none;
}
```

## CSS Classes

### Slots

- `popover__arrow`: The class for the arrow slot.
- `popover__backdrop`: The class for the backdrop slot.
- `popover__close`: The class for the close slot.
- `popover__content`: The class for the content slot.
- `popover__description`: The class for the description slot.
- `popover__popup`: The class for the popup slot.
- `popover__portal`: The class for the portal slot.
- `popover__positioner`: The class for the positioner slot.
- `popover`: The class for the root slot.
- `popover__title`: The class for the title slot.
- `popover__trigger`: The class for the trigger slot.
- `popover__viewport`: The class for the viewport slot.
