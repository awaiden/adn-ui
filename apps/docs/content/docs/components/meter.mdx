## Styling

### Customizing the component classes

```css
.meter {
  @apply w-full;
}

.meter__track {
  @apply bg-secondary relative h-2 w-full overflow-hidden rounded-full;
}

.meter__indicator {
  @apply bg-primary h-full w-full flex-1 transition-all;
}
```

## CSS Classes

### Slots

- `meter__indicator`: The class for the indicator slot.
- `meter__label`: The class for the label slot.
- `meter`: The class for the root slot.
- `meter__track`: The class for the track slot.
- `meter__value`: The class for the value slot.

### Size Modifiers

- `meter--lg`: Size lg modifier.
- `meter--md`: Size md modifier.
- `meter--sm`: Size sm modifier.

### Variant Modifiers

- `meter--danger`: Variant danger modifier.
- `meter--primary`: Variant primary modifier.
- `meter--secondary`: Variant secondary modifier.
- `meter--success`: Variant success modifier.
