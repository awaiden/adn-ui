## Styling

### Customizing the component classes

```css
.number-field {
  @apply flex flex-col gap-2;
}

.number-field__input {
  @apply border-input placeholder:text-muted-foreground focus-visible:ring-ring flex h-9 w-full rounded-md border bg-transparent px-3 py-1 text-center shadow-sm transition-colors file:border-0 file:bg-transparent file:text-sm file:font-medium focus-visible:ring-1 focus-visible:outline-none disabled:cursor-not-allowed disabled:opacity-50;
}
```

## CSS Classes

### Slots

- `number-field__decrement`: The class for the decrement slot.
- `number-field__group`: The class for the group slot.
- `number-field__increment`: The class for the increment slot.
- `number-field__input`: The class for the input slot.
- `number-field`: The class for the root slot.
- `number-field__scrub-area`: The class for the scrubArea slot.
- `number-field__scrub-area-cursor`: The class for the scrubAreaCursor slot.
