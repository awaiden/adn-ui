## CSS Classes

### Slots

- `select`: The class for the root slot.
- `select__trigger`: The class for the trigger slot.
- `select__value`: The class for the value slot.
- `select__icon`: The class for the icon slot.
- `select__portal`: The class for the portal slot.
- `select__backdrop`: The class for the backdrop slot.
- `select__positioner`: The class for the positioner slot.
- `select__popup`: The class for the popup slot.
- `select__arrow`: The class for the arrow slot.
- `select__list`: The class for the list slot.
- `select__item`: The class for the item slot.
- `select__item-text`: The class for the itemText slot.
- `select__item-indicator`: The class for the itemIndicator slot.
- `select__separator`: The class for the separator slot.
- `select__group`: The class for the group slot.
- `select__group-label`: The class for the groupLabel slot.
