## Styling

### Customizing the component classes

```css
.fieldset {
  @apply bg-card text-card-foreground rounded-xl border p-6 shadow;
}

.fieldset__legend {
  @apply -ml-1 text-sm font-medium;
}
```

## CSS Classes

### Slots

- `fieldset__legend`: The class for the legend slot.
- `fieldset`: The class for the root slot.
