## Styling

### Passing Tailwind Classes

```tsx
<Button className="bg-red-500 text-white hover:bg-red-600">Red Button</Button>
```

### Customizing the component classes

```css
.button {
  @apply bg-blue-500 text-white hover:bg-blue-600;
}

.button--secondary {
  @apply bg-gray-500 text-white hover:bg-gray-600;
}
```

## CSS Classes

### Base Class

- `button`: The base class for the button component.

### Size Modifiers

- `button--sm`: Small size modifier.
- `button--md`: Medium size modifier.
- `button--lg`: Large size modifier.

### Variant Modifiers

- `button--primary`: Primary variant modifier.
- `button--secondary`: Secondary variant modifier.
- `button--tertiary`: Tertiary variant modifier.
- `button--ghost`: Ghost variant modifier.
- `button--danger`: Danger variant modifier.
