## Styling

### Customizing the component classes

```css
.slider {
  @apply relative flex w-full touch-none items-center select-none;
}

.slider__track {
  @apply bg-primary/20 relative h-1.5 w-full grow overflow-hidden rounded-full;
}

.slider__range {
  @apply bg-primary absolute h-full;
}

.slider__thumb {
  @apply border-primary/50 bg-background focus-visible:ring-ring block h-4 w-4 rounded-full border shadow transition-colors focus-visible:ring-1 focus-visible:outline-none disabled:pointer-events-none disabled:opacity-50;
}
```

## CSS Classes

### Slots

- `slider__control`: The class for the control slot.
- `slider__indicator`: The class for the indicator slot.
- `slider`: The class for the root slot.
- `slider__thumb`: The class for the thumb slot.
- `slider__track`: The class for the track slot.
- `slider__value`: The class for the value slot.
