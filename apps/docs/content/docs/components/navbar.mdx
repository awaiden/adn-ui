## Styling

### Customizing the component classes

```css
.navbar {
  @apply bg-background sticky top-0 z-40 w-full border-b;
}
```

## CSS Classes

### Slots

- `navbar__container`: The class for the container slot.
- `navbar__content`: The class for the content slot.
- `navbar__list`: The class for the list slot.
- `navbar__list-item`: The class for the listItem slot.
- `navbar__menu`: The class for the menu slot.
- `navbar__menu-item`: The class for the menuItem slot.
- `navbar`: The class for the root slot.
- `navbar__toggle`: The class for the toggle slot.
