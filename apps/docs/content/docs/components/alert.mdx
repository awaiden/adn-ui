## Styling

### Customizing the component classes

```css
.alert {
  @apply [&>svg]:text-foreground relative w-full rounded-lg border px-4 py-3 text-sm [&>svg]:absolute [&>svg]:top-4 [&>svg]:left-4 [&>svg+div]:translate-y-[-3px] [&>svg~*]:pl-7;
}

.alert--default {
  @apply bg-background text-foreground;
}

.alert--destructive {
  @apply border-destructive/50 text-destructive dark:border-destructive [&>svg]:text-destructive;
}
```

## CSS Classes

### Slots

- `alert__content`: The class for the content slot.
- `alert__description`: The class for the description slot.
- `alert__indicator`: The class for the indicator slot.
- `alert`: The class for the root slot.
- `alert__title`: The class for the title slot.

### Variant Modifiers

- `alert--danger`: Variant danger modifier.
- `alert--info`: Variant info modifier.
- `alert--primary`: Variant primary modifier.
- `alert--success`: Variant success modifier.
- `alert--warning`: Variant warning modifier.
