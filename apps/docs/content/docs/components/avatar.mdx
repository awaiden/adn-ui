## Styling

### Customizing the component classes

```css
.avatar {
  @apply relative flex h-10 w-10 shrink-0 overflow-hidden rounded-full;
}

.avatar__image {
  @apply aspect-square h-full w-full;
}

.avatar__fallback {
  @apply bg-muted flex h-full w-full items-center justify-center rounded-full;
}
```

## CSS Classes

### Slots

- `avatar__fallback`: The class for the fallback slot.
- `avatar__image`: The class for the image slot.
- `avatar`: The class for the root slot.

### Size Modifiers

- `avatar--lg`: Size lg modifier.
- `avatar--md`: Size md modifier.
- `avatar--sm`: Size sm modifier.
