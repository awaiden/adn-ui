## Styling

### Customizing the component classes

```css
.card {
  @apply bg-card text-card-foreground rounded-xl border shadow;
}

.card__header {
  @apply flex flex-col space-y-1.5 p-6;
}

.card__title {
  @apply leading-none font-semibold tracking-tight;
}

.card__description {
  @apply text-muted-foreground text-sm;
}

.card__content {
  @apply p-6 pt-0;
}

.card__footer {
  @apply flex items-center p-6 pt-0;
}
```

## CSS Classes

### Slots

- `card__content`: The class for the content slot.
- `card__description`: The class for the description slot.
- `card__header`: The class for the header slot.
- `card`: The class for the root slot.
- `card__title`: The class for the title slot.
