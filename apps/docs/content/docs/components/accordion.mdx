## Styling

### Customizing the component classes

```css
.accordion {
  @apply w-full;
}

.accordion__item {
  @apply border-b border-gray-200 dark:border-gray-800;
}

.accordion__header {
  @apply flex;
}

.accordion__trigger {
  @apply flex flex-1 items-center justify-between py-4 font-medium transition-all hover:underline [&[data-state=open]>svg]:rotate-180;
}

.accordion__trigger-icon {
  @apply h-4 w-4 shrink-0 transition-transform duration-200;
}

.accordion__panel {
  @apply data-[state=closed]:animate-accordion-up data-[state=open]:animate-accordion-down overflow-hidden text-sm transition-all;
}

.accordion__content {
  @apply pt-0 pb-4;
}
```

## CSS Classes

### Slots

- `accordion__content`: The class for the content slot.
- `accordion__header`: The class for the header slot.
- `accordion__item`: The class for the item slot.
- `accordion__panel`: The class for the panel slot.
- `accordion`: The class for the root slot.
- `accordion__trigger`: The class for the trigger slot.
- `accordion__trigger-icon`: The class for the triggerIcon slot.
