## Styling

### Customizing the component classes

```css
.list {
  @apply flex flex-col gap-2;
}

.list__item {
  @apply flex items-center gap-2;
}
```

## CSS Classes

### Slots

- `list__item`: The class for the item slot.
- `list`: The class for the root slot.
