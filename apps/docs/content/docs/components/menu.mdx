## Styling

### Customizing the component classes

```css
.menu__popup {
  @apply bg-popover text-popover-foreground z-50 min-w-[8rem] overflow-hidden rounded-md border p-1 shadow-md transition-all;
}

.menu__item {
  @apply focus:bg-accent focus:text-accent-foreground relative flex cursor-default items-center rounded-sm px-2 py-1.5 text-sm transition-colors outline-none select-none data-[disabled]:pointer-events-none data-[disabled]:opacity-50;
}
```

## CSS Classes

### Slots

- `menu__arrow`: The class for the arrow slot.
- `menu__backdrop`: The class for the backdrop slot.
- `menu__checkbox-item`: The class for the checkboxItem slot.
- `menu__group`: The class for the group slot.
- `menu__group-label`: The class for the groupLabel slot.
- `menu__item`: The class for the item slot.
- `menu__popup`: The class for the popup slot.
- `menu__portal`: The class for the portal slot.
- `menu__positioner`: The class for the positioner slot.
- `menu__radio-group`: The class for the radioGroup slot.
- `menu__radio-item`: The class for the radioItem slot.
- `menu`: The class for the root slot.
- `menu__separator`: The class for the separator slot.
- `menu__submenu`: The class for the submenu slot.
- `menu__submenu__trigger`: The class for the submenuTrigger slot.
- `menu__trigger`: The class for the trigger slot.
