## Styling

### Customizing the component classes

```css
.alert-dialog__backdrop {
  @apply data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=closed]:fade-out-0 data-[state=open]:fade-in-0 fixed inset-0 z-50 bg-black/80;
}

.alert-dialog__popup {
  @apply bg-background data-[state=open]:animate-in data-[state=closed]:animate-out data-[state=closed]:fade-out-0 data-[state=open]:fade-in-0 data-[state=closed]:zoom-out-95 data-[state=open]:zoom-in-95 data-[state=closed]:slide-out-to-left-1/2 data-[state=closed]:slide-out-to-top-[48%] data-[state=open]:slide-in-from-left-1/2 data-[state=open]:slide-in-from-top-[48%] fixed top-[50%] left-[50%] z-50 grid w-full max-w-lg translate-x-[-50%] translate-y-[-50%] gap-4 border p-6 shadow-lg duration-200 sm:rounded-lg;
}
```

## CSS Classes

### Slots

- `alert-dialog__backdrop`: The class for the backdrop slot.
- `alert-dialog__close`: The class for the close slot.
- `alert-dialog__description`: The class for the description slot.
- `alert-dialog__popup`: The class for the popup slot.
- `alert-dialog__portal`: The class for the portal slot.
- `alert-dialog`: The class for the root slot.
- `alert-dialog__title`: The class for the title slot.
- `alert-dialog__trigger`: The class for the trigger slot.
- `alert-dialog__viewport`: The class for the viewport slot.
