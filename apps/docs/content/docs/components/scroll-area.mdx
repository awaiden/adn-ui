## Styling

### Customizing the component classes

```css
.scroll-area {
  @apply relative overflow-hidden;
}

.scroll-area__viewport {
  @apply h-full w-full rounded-[inherit];
}

.scroll-area__scrollbar {
  @apply flex touch-none transition-colors select-none;
}

.scroll-area__thumb {
  @apply bg-border relative flex-1 rounded-full;
}
```

## CSS Classes

### Slots

- `scroll-area`: The class for the root slot.
- `scroll-area__viewport`: The class for the viewport slot.
- `scroll-area__content`: The class for the content slot.
- `scroll-area__scrollbar`: The class for the scrollbar slot.
- `scroll-area__thumb`: The class for the thumb slot.
- `scroll-area__corner`: The class for the corner slot.
