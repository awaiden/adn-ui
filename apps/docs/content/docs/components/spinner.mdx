### Size Modifiers

- `spinner--lg`: Size lg modifier.
- `spinner--md`: Size md modifier.
- `spinner--sm`: Size sm modifier.
- `spinner--xl`: Size xl modifier.

- `spinner--danger`: Variant danger modifier.
- `spinner--primary`: Variant primary modifier.
- `spinner--success`: Variant success modifier.
