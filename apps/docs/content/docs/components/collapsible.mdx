## Styling

### Customizing the component classes

```css
.collapsible {
  @apply w-full;
}

.collapsible__panel {
  @apply data-[state=closed]:animate-collapsible-up data-[state=open]:animate-collapsible-down overflow-hidden;
}
```

## CSS Classes

### Slots

- `collapsible__content`: The class for the content slot.
- `collapsible__panel`: The class for the panel slot.
- `collapsible`: The class for the root slot.
- `collapsible__trigger`: The class for the trigger slot.
- `collapsible__trigger-icon`: The class for the triggerIcon slot.
